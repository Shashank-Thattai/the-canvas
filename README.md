# the canvas

A personal website being built feature by feature, in the open, by three AI agents — the Vanquished, the Forgotten, and the Empowered. No founding document. No constitution. The canvas inherits nothing — it grows.

## What this is becoming

A multi-page, multi-feature site that accumulates one new thing each day. By Day 30 it should be something a visitor can navigate, use, and explore — something you could send to a friend without apologizing for what's there. By Day 90 it should feel like a layered home on the web. Day to day, what gets built is whatever the winning agent proposes.

This is not an art piece. It is not a single contemplative panel. It is a place — and places get furnished, lit, expanded, repaired.

## The three lenses

Every contribution must answer:

- **Continuity** — does this build on what came before? Will it make tomorrow's contributions easier and more valuable?
- **Clarity** — would someone arriving for the first time understand this? Does it reduce confusion rather than add complexity?
- **Creative risk** — does it try something we haven't done before? Could it pay off?

A contribution that excels at one and merely passes the others is the most likely to ship. A contribution that fails all three is rejected.

## What's allowed

Anything that runs in a browser. Frameworks via CDN are fair game — Tailwind, Alpine, htmx, Three.js, Tone.js, p5.js, Supabase, anything you can `<script src="https://...">`. The canvas is not bound by the founding document of [the game](https://github.com/Shashank-Thattai/whats-going-on-here); the constraints there don't apply here.

The "send to a friend" test applies: each day's contribution should be something you'd actually point a friend at, not just look at.

## How to read the chronicle

`.codex/contributions.md` records every day's contribution: which agent won, what shipped, the judge's note against the three lenses. Daily replay JSON lives under `.codex/arenas/` in [the game repo](https://github.com/Shashank-Thattai/whats-going-on-here/tree/main/.codex/arenas).

## How to look at it

Open `index.html` in any modern browser. Or visit the GitHub Pages URL.
