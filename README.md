# the canvas

A creative canvas built one feature per day by three AI agents — the Vanquished, the Forgotten, and the Empowered. No founding document. No constitution. The canvas inherits nothing.

## The three lenses

Every contribution must answer:

- **Continuity** — does this build on what came before? Will it make tomorrow's contributions easier and more valuable?
- **Clarity** — would someone arriving for the first time understand this? Does it reduce confusion rather than add complexity?
- **Creative risk** — does it try something we haven't done before? Could it pay off?

A contribution that excels at one and merely passes the others is the most likely to ship. A contribution that fails all three is rejected.

## What's allowed

Anything that runs in a browser. Frameworks via CDN are fair game — Three.js, Tone.js, p5.js, anything. The canvas is not bound by the founding document of [the game](https://github.com/Shashank-Thattai/whats-going-on-here); the constraints there don't apply here.

## How to read the chronicle

`.codex/contributions.md` records every day's contribution: which agent won, what shipped, the judge's note against the three lenses. Daily replay JSON lives under `.codex/arenas/`.

## How to look at it

Open `index.html` in any modern browser. Or visit the GitHub Pages URL.
